# nestofindia

Official website for **Nest of India** — the national council for the Indian tattoo and art community.

## Stack

- [Next.js 14](https://nextjs.org/) (App Router)
- React 18 + TypeScript
- Tailwind CSS

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Production build

```bash
npm run build
npm start
```

## Deploy

This repo is ready to deploy on [Vercel](https://vercel.com/) or any Node.js host that supports Next.js. Connect the GitHub repository and use the default Next.js build settings.
